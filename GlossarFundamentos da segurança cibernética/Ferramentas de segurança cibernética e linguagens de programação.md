# 📖 Glossário: Ferramentas e Processos Técnicos
# Módulo:  Ferramentas de segurança cibernética e linguagens de programação

Este repositório foca nas ferramentas de monitoramento, linguagens de dados e procedimentos de preservação de evidências digitais.

---

## 🛠️ Ferramentas de Monitoramento e Defesa

Softwares essenciais para a detecção de ameaças e resposta a incidentes.

| Ferramenta | Descrição |
| :--- | :--- |
| **SIEM** | *Security Information and Event Management*. Aplicativo que coleta e analisa dados de registros (logs) para monitorar atividades críticas. |
| **IDS** | *Intrusion Detection System*. Aplicativo que monitora a atividade do sistema e gera alertas sobre possíveis intrusões. |
| **Antivírus** | Software especializado em prevenir, detectar e eliminar malwares e vírus de dispositivos. |
| **Packet Sniffer** | Analisador de protocolo de rede projetado para capturar e analisar o tráfego de dados em tempo real. |

---

## 📊 Dados e Programação

Conceitos relacionados à manipulação e armazenamento de informações.

* **Dados:** Um pedaço específico de informação.
* **Banco de Dados:** Uma coleção organizada de informações ou dados estruturados.
* **SQL (Structured Query Language):** Linguagem de consulta usada para criar, interagir e solicitar informações de bancos de dados.
* **Programação:** Processo de criação de um conjunto específico de instruções para que um computador execute tarefas.
* **Linux:** Sistema operacional de código aberto, amplamente utilizado em servidores e ferramentas de segurança.

---

## 🔍 Registros e Forense Digital

Processos cruciais para a análise pós-incidente e conformidade.



* **Geração de Registros (Logging):** O ato de criar um registro de eventos (Log) que ocorrem nos sistemas de uma organização.
* **Ordem de Volatilidade:** Uma sequência técnica que descreve quais dados devem ser preservados primeiro (do mais volátil, como a memória RAM, ao menos volátil, como o disco rígido).
* **Proteção e Preservação de Evidências:** O processo de manusear adequadamente evidências digitais que podem ser frágeis ou voláteis para garantir sua integridade em investigações.
