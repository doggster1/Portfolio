# 🛡️ Glossário: Connect and Protect: Networks and Network Security
# Módulo: Fortalecimento da segurança

| Termo | Definição |
| :--- | :--- |
| **Configuração da Linha de Base (Baseline)** | Um conjunto documentado de especificações em um sistema que serve como padrão para futuras compilações, liberações e atualizações. |
| **Hardware** | Os componentes físicos e tangíveis de um computador. |
| **Autenticação Multifator (MFA)** | Medida de segurança que exige que o usuário verifique sua identidade de duas ou mais formas independentes para acessar um sistema. |
| **Análise de Logs de Rede** | O processo de examinar registros de rede para identificar eventos de interesse, anomalias ou falhas. |
| **Sistema Operacional (SO)** | O software que atua como interface entre o hardware do computador e o usuário/aplicativos. |
| **Atualização de Patch** | Atualização de software ou sistema operacional que corrige vulnerabilidades de segurança ou bugs em um produto. |
| **Teste de Penetração (Pen Test)** | Um ataque simulado e autorizado para identificar vulnerabilidades em sistemas, redes, sites e processos. |
| **Hardening (Fortalecimento)** | O processo de configurar e reforçar um sistema para reduzir suas vulnerabilidades e minimizar a superfície de ataque. |
| **SIEM (Gerenciamento de Eventos e Informações)** | Aplicativo que centraliza, coleta e analisa dados de registros para monitorar atividades críticas de uma organização. |
| **Arquivo Gravável Globalmente** | Um arquivo com permissões excessivas que permitem que qualquer usuário do sistema possa alterá-lo (*World-writable*). |
