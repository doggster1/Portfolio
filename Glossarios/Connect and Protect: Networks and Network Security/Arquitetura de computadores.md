# Glossário: Fundamentos de Rede
# Módulo: Arquitetura de computadores

Este glossário contém os termos essenciais para a compreensão de infraestrutura de rede e comunicação de dados.

---

### A - C
* **Largura de banda:** A capacidade máxima de transmissão de dados em uma rede, medida em bits por segundo (bps).
* **Computação em nuvem:** A prática de usar servidores remotos, aplicativos e serviços de rede hospedados na Internet em vez de em dispositivos físicos locais.
* **Rede de computadores em nuvem:** Uma coleção de servidores ou computadores que armazenam recursos e dados em data centers remotos que podem ser acessados pela Internet.

### D - I
* **Pacote de dados:** Uma unidade básica de informações que viaja de um dispositivo para outro em uma rede.
* **Hub:** Um dispositivo de rede que transmite informações para todos os dispositivos da rede de forma indiscriminada.
* **Protocolo de Internet (IP):** Um conjunto de padrões usados para roteamento e endereçamento de pacotes de dados à medida que eles trafegam entre dispositivos em uma rede.
* **Endereço IP:** Uma sequência única de caracteres que identifica a localização de um dispositivo na Internet.

### L - M
* **LAN (Local Area Network):** Uma rede que abrange áreas pequenas, como um prédio de escritórios, uma escola ou uma residência.
* **Endereço MAC (Media Access Control):** Um identificador alfanumérico único atribuído a cada dispositivo físico (placa de rede) em uma rede.
* **Modem:** Um dispositivo que conecta o roteador à Internet, convertendo sinais para permitir o acesso à rede externa.
* **Modelo OSI (Open Systems Interconnection):** Um conceito padronizado que descreve as **sete camadas** que os computadores usam para se comunicar e enviar dados.

### N - P
* **Rede:** Um grupo de dispositivos conectados entre si para compartilhar recursos.
* **Interceptação de pacotes (Packet Sniffing):** A prática de capturar e inspecionar pacotes de dados que trafegam em uma rede.
* **Porta:** Um local baseado em software que organiza o envio e o recebimento de dados entre aplicações em uma rede.

### R - S
* **Roteador:** Um dispositivo de rede que conecta várias redes diferentes (ex: sua casa à Internet) e decide o melhor caminho para os dados.
* **Velocidade:** A taxa real na qual um dispositivo envia e recebe dados, medida em bits por segundo.
* **Switch:** Um dispositivo que cria conexões diretas entre dispositivos específicos em uma rede local, enviando dados apenas ao destinatário correto.

### T - W
* **Modelo TCP/IP:** A estrutura principal da Internet usada para visualizar como os dados são organizados e transmitidos.
* **Protocolo TCP (Transmission Control Protocol):** Um protocolo orientado à conexão que garante que os dados cheguem na ordem correta e sem erros.
* **User Datagram Protocol (UDP):** Um protocolo sem conexão, focado em velocidade, que não verifica se os dados foram recebidos.
* **WAN (Wide Area Network):** Uma rede que abrange uma grande área geográfica, como cidades, estados ou países.
